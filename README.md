# ProjectTemplate

## Setup
- clone git repo
- open with for example visual studio code
- run 'npm init'
- use mongoDB with localhost
-- https://docs.mongodb.com/manual/tutorial/enable-authentication/ 
- create .env file and add fields for
-- DB_USER=username
-- DB_PASS=password
-- DB_HOST=localhost
-- DB_PORT=27017
-- DB_NAME=database
-- SESSION_SEC=randomstring
